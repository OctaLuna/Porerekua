# Porerekua — ser solidario, compartir lo que se tiene

Este proyecto se llama **Porerekua**, que significa "ser solidario, compartir lo que se tiene". Es una plataforma dedicada a la conservación, conocimiento y colaboración en torno a los bosques y su gente.

## Ejecutar localmente

**Requisitos previos:** Node.js

1. Instala las dependencias:
   `npm install`
2. Configura tu `GEMINI_API_KEY` en [.env.local](.env.local)
3. Ejecuta la app:
   `npm run dev`
